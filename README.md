## Description

This is the backend for Kallos, an online Greek gifts store made with Angular, TypeScript, and Nest.js.

Frontend Repository : [nawresncib12/Kallos](https://github.com/nawresncib12/Kallos)

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
